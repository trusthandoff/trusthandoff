# Security Policy

## Official sources

The official TrustHandoff repository is:

- `https://github.com/trusthandoff/trusthandoff`

Official package publication is performed through GitHub Actions Trusted Publishing to PyPI from the official repository only.

## Supported versions

| Version | Supported |
|---------|-----------|
| 0.x     | Yes       |

## Reporting a vulnerability

Please do **not** open a public GitHub issue for security vulnerabilities.

Report vulnerabilities privately through GitHub Security Advisories or contact the maintainer directly.

Preferred channel:
- GitHub Security Advisory (private report)

If private contact is needed, use:
- `266288756+shaymizuno@users.noreply.github.com`

## Scope

Please report issues related to:
- signature verification
- delegation validation
- capability/token handling
- trust boundaries between agents
- publishing / supply-chain integrity

## Disclosure policy

We will investigate credible reports and aim to acknowledge receipt promptly.
Please allow time for triage before public disclosure.
